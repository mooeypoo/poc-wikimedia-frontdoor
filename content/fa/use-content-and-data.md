# یادگیری

## آموزش‌ها و راهنماها

مستندات کامل برای کار با API REST ویکی‌مدیا را کاوش کنید. موضوعات شامل احراز هویت با OAuth 2.0، صفحه‌بندی نتایج، مدیریت محتوای چندزبانه با پشتیبانی کامل از یونیکد و متن دوجهته، و نوشتن کلاینت‌های API مقاوم است که محدودیت‌های نرخ را رعایت می‌کنند. با راهنمای شروع سریع آغاز کنید و سپس از کاوشگر API برای اجرای درخواست‌های زنده استفاده کنید.
