# جامعه

## به جامعه بپیوندید

پروژه API ویکی‌مدیا توسط داوطلبان و کارمندان بنیاد ویکی‌مدیا توسعه می‌یابد. به لیست پستی توسعه‌دهندگان ویکی‌مدیا بپیوندید، مشکلات را در Phabricator گزارش دهید و در جلسات مجازی دفتر API شرکت کنید. روش‌های زیادی برای مشارکت وجود دارد: از مستندسازی نقاط پایانی و ترجمه راهنماها تا بررسی تغییرات کد.
