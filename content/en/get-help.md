# Get help

## Troubleshooting and support

If your API requests are being throttled or returning unexpected error codes, consult the rate limit and quota documentation before filing a support request. This section provides debugging guides for common issues, including authentication failures, malformed request headers, and unexpected response shapes. For persistent problems, you can reach the API team through the developer mailing list or by opening a task in Wikimedia Phabricator with detailed reproduction steps.
