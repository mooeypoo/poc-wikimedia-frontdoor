# Build on-wki features

On-wiki features make it easier for editors and readers to use and maintain Wikimedia projects.

## MediaWiki frontend API

Interact with MediaWiki from user scripts, gadgets, skins, and extensions.

- [Read the docs](https://doc.wikimedia.org/mediawiki-core/master/js/)

## MediaWiki and extensions

MediaWiki is the software that powers Wikipedia, its sister projects, and thousands of wikis all over the world.

- [Read the MediaWiki developer documentation](https://www.mediawiki.org/wiki/Special:MyLanguage/Developer_hub)
- [Discover extensions](https://www.mediawiki.org/wiki/Special:MyLanguage/Manual:Extensions)

## Learn with tutorials

### Add JavaScript or CSS to a wiki

Customize MediaWiki with client-side JavaScript and CSS. Share user scripts, and explore gadgets other people have created.

- [Read more on mediawiki.org](https://www.mediawiki.org/wiki/Special:MyLanguage/Gadget_kitchen)
