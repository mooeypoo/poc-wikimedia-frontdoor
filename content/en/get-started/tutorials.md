# Tutorials

Follow step-by-step guides to learn the basics of building with Wikimedia.

## Featured tutorials

### Display the picture of the day

Learn how to use the MediaWiki API to fetch the daily featured picture from Wikipedia. This tutorial uses Python and the Flask framework.

- [Read more on mediawiki.org](https://www.mediawiki.org/wiki/Special:MyLanguage/API:Picture_of_the_day_viewer)

### Host a tool on Wikimedia cloud servers

Toolforge is a free hosting environment for running and managing tools. Get started with Toolforge by creating an account and setting up a basic web service.

- [Read more on Wikitech](https://wikitech.wikimedia.org/wiki/Help:Toolforge/Quickstart)

## Browse all tutorials

### Build your first tool

Learn how to edit wiki pages using Python and the Pywikibot framework. This tutorial helps you get started quickly using only your browser.

- [Read more on mediawiki.org](https://www.mediawiki.org/wiki/Special:MyLanguage/Manual:Pywikibot/PAWS)

### Display holidays from Wikipedia

Learn how to use the MediaWiki API to fetch holidays for a given date from Wikipedia. This tutorial also covers how to use the API to log in to Wikipedia and add new holidays. This tutorial uses Python and the Flask framework.

- [Read more on mediawiki.org](https://www.mediawiki.org/wiki/Special:MyLanguage/API:Holidays_viewer)

### Display information from Wikidata

Use the MediaWiki API to fetch and display information from Wikidata. Learn how Wikipedia articles are associated with Wikidata and how to parse Wikidata structures.

- [Read more on mediawiki.org](https://www.mediawiki.org/wiki/Special:MyLanguage/API:Presenting_Wikidata_knowledge)

### Display the picture of the day

Learn how to use the MediaWiki API to fetch the daily featured picture from Wikipedia. This tutorial uses Python and the Flask framework.

- [Read more on mediawiki.org](https://www.mediawiki.org/wiki/Special:MyLanguage/API:Picture_of_the_day_viewer)

### Generate ideas for articles

Learn how to use the MediaWiki API to generate ideas for new Wikipedia articles. This tutorial uses Python and the Flask framework.

- [Read more on mediawiki.org](https://www.mediawiki.org/wiki/Special:MyLanguage/API:Article_ideas_generator)

### Get started querying Wikidata

Learn how to query Wikidata with SPARQL. This tutorial covers the basics of Wikidata and how to use the Wikidata Query Service.

- [Read more](https://wdqs-tutorial.toolforge.org)

### Search articles by location

Learn how to use the MediaWiki API to search for Wikipedia articles near a specific location. This tutorial uses JavaScript, Python, and the Flask framework.

- [Read more on mediawiki.org](https://www.mediawiki.org/wiki/Special:MyLanguage/API:Nearby_places_viewer)

### Write a bot that uses Wikidata

Use Python to handle large datasets from Wikidata and transfer data between tools in your workflow.

- [Read more on Wikidata](https://www.wikidata.org/wiki/Special:MyLanguage/Wikidata:Pywikibot_-_Python_3_Tutorial)

### Add JavaScript or CSS to a wiki

Customize MediaWiki with client-side JavaScript and CSS. Share user scripts, and explore gadgets other people have created.

- [Read more on mediawiki.org](https://www.mediawiki.org/wiki/Special:MyLanguage/Gadget_kitchen)

### Host a tool on Wikimedia cloud servers

Toolforge is a free hosting environment for running and managing tools. Get started with Toolforge by creating an account and setting up a basic web service.

- [Read more on Wikitech](https://wikitech.wikimedia.org/wiki/Help:Toolforge/Quickstart)
