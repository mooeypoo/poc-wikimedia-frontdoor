# Learn

## Tutorials and guides

This section contains step-by-step tutorials and reference guides for working with the Wikimedia REST API. Topics include authenticating with OAuth 2.0, paginating through large result sets, handling multilingual content with proper Unicode and bidirectional text support, and writing resilient API clients that respect throttling and rate limits. Start with the getting started guide, then explore the API Explorer to run live queries against real Wikimedia endpoints.
