# למידה

## מדריכים והדרכות

עיינו בתיעוד המקיף לעבודה עם ה-API של ויקימדיה. הנושאים כוללים אימות עם OAuth 2.0, עימוד תוצאות, טיפול בתוכן רב-לשוני עם תמיכה מלאה ב-Unicode וטקסט דו-כיווני, וכתיבת לקוחות API עמידים המכבדים את מגבלות הקצב. התחילו עם מדריך ההתחלה המהירה ואז חקרו את סייר ה-API להרצת שאילתות חיות מול שרתי ויקימדיה האמיתיים.
