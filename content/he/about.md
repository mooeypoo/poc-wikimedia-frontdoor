## המשימה שלנו

Front Door נבנה ומתוחזק על ידי קרן ויקימדיה כדי לתמוך במפתחים הבונים על תשתית ויקימדיה הפתוחה. הפורטל מתעד את פני ה-REST API הזמינים לציבור בדיוק ובשקיפות. הוא חלק ממחויבות הקרן לידע פתוח: אותו תוכן שמניע את ויקיפדיה ופרויקטי אחיותיה נגיש בחינם דרך ממשקי API יציבים ומתועדים היטב.
