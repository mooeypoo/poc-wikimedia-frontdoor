## A propos de ce portail

Le Portail des developpeurs Wikimedia est la ressource centrale pour creer des applications avec les API REST de Wikimedia. Il couvre tous les projets Wikimedia — Wikipedia, Wikidata, Wikimedia Commons et plus encore. Que vous creez un pipeline d'analyse de donnees, une visualisation interactive ou une extension de navigateur, vous trouverez ici la documentation des endpoints, les guides d'authentification et les bacs a sable d'API en direct.
