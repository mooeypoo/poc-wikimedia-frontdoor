## Acerca de este portal

El Portal de Desarrolladores de Wikimedia es el recurso central para crear aplicaciones con las API REST de Wikimedia. Cubre todos los proyectos de Wikimedia — Wikipedia, Wikidata, Wikimedia Commons y mas. Ya sea que estes creando un proceso de analisis de datos, una visualizacion interactiva o una extension de navegador, aqui encontraras documentacion de endpoints, guias de autenticacion y entornos de prueba de API en vivo.
