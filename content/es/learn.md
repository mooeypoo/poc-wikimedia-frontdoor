# Aprender

## Tutoriales y guías

Explora la documentación completa para trabajar con la API REST de Wikimedia. Los temas incluyen autenticación con OAuth 2.0, paginación de resultados, manejo de contenido multilingüe con soporte adecuado de texto bidireccional y Unicode, y escritura de clientes de API resilientes que respetan los límites de velocidad. Comienza con la guía de inicio rápido y luego usa el Explorador de API para ejecutar consultas en vivo contra los servidores reales de Wikimedia.
